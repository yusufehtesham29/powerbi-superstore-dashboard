# Superstore Sales Performance Dashboard
📊 Interactive Power BI dashboard analyzing Superstore sales (2014–2017) by region, category, and product. Includes KPIs, trends, and filters.

-> Overview
This repository contains an interactive Power BI dashboard analyzing Superstore sales (2014–2017). The dashboard provides insights using KPIs, trend analyses, sales by product, region, category, and more.

-> Files Included
- `Superstore Sales Performance Dashboard.pbix` – Power BI dashboard file
- `Sample - Superstore.csv` – Original dataset (open-source)

-> Features
- Dynamic KPIs: Sales, Profit, Profit Margin, Quantity, Discounts
- Trend analysis by year
- Top products chart
- Regional and category breakdown
- Interactivity via slicers (Year, Region, etc.)

-> Dataset Source
[Superstore Dataset on Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final?resource=download)

-> How to Use
1. Download the `.pbix` file
2. Make sure the `.csv` file is in the same folder (if required by Power BI)
3. Open in Power BI Desktop
